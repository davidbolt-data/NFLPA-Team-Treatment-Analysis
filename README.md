# NFLPA-Team-Treatment-Analysis
NFLPA Team Treatment Analysis

Project Overview
This project analyzes NFL Players Association report card data to evaluate how teams treat players across categories such as facilities, nutrition programs, travel accommodations, and family support. The goal is to identify which organizations provide the best player environments and what factors most influence overall team satisfaction.

Business Question
Which NFL teams score highest in player satisfaction, and what factors contribute most to positive player experiences?

Dataset
NFLPA Team Report Card data containing grades for multiple organizational categories.

Tools Used
Python
Pandas
Tableau
Excel

Key Insights
Teams with higher facility grades consistently rank higher overall.
Nutrition programs show strong correlation with overall team satisfaction.
Some teams perform well in coaching but poorly in travel and facilities, highlighting operational gaps.
